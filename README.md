# JavaScript UI Components

A simple project demonstrating interactive UI components built using **HTML5**, **CSS3**, and **JavaScript**.

The project focuses on JavaScript integration by adding interactivity to reusable UI components.

---

## 🚀 Features

- Responsive Navbar
- Mobile Navigation Toggle
- Modal Open & Close
- Interactive Tabs
- Responsive Design
- Organized JavaScript Functions

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
